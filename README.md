# Shaolink CLI

> URL shortener from terminal

[![License](https://img.shields.io/npm/l/inita-cli.svg)](https://github.com/Akevic/inita-cli/blob/master/package.json)

## Install

```
$ npm install -g shaolink-cli
```

## Usage

```
$ slk <url>
```
